## 📖 游戏介绍

### 来自蝴蝶忍的一封信

初次见面的弟子你好，我是**虫柱蝴蝶忍**。让我为你介绍这座训练场的全部内容~

这是一款**回合制线上 PVP 对战游戏**，你将扮演鬼杀队剑士或十二鬼月，在 6×12 的网格战场上对决。每回合你可以选择**移动一格**或**释放一个技能**，双方行动同时结算。击败对手即可获胜——但请记住，鬼的弱点在于脖颈，精准命中才是斩杀的关键。

### 核心特色

| 特色 | 说明 |
|------|------|
| 👥 **双人对战** | 创建房间 → 分享6位房间码 → 选角色选技能 → 开战 |
| 🤖 **单人模式** | AI 对手陪练，🌱简单 / ⚔️普通 / 💀困难 三级可选 |
| 🦋 **AI 向导** | 我（蝴蝶忍）会根据你的战斗数据给出个性化指导 |
| 📊 **战斗记录** | 每场完整数据自动保存本地，支持导出 JSON 训练模型 |
| 📱 **响应式** | 桌面端 + 手机端均可流畅对战 |
| 🔌 **断线重连** | 掉线自动重连，房间保留10分钟 |

### 角色

**人方（鬼杀队）14 名角色**：灶门炭治郎、富冈义勇、我妻善逸、嘴平伊之助、炼狱杏寿郎、宇髓天元、时透无一郎、甘露寺蜜璃、不死川实弥、伊黑小芭内、悲鸣屿行冥、栗花落香奈乎、不死川玄弥、灶门祢豆子

**鬼方 7 名角色**：累、魇梦、妓夫太郎&堕姬、猗窝座、童磨、黑死牟、半天狗

> 目前 MVP 版本实装了**炭治郎**（水之呼吸6选3）和**猗窝座**（3技能），更多角色逐步解锁中~

### 技能机制

| 效果 | 含义 |
|------|------|
| AD | 攻击判定 |
| AS | 位移技能 |
| DB | 障碍破除 |
| CAM | 伤害+位移双控 |
| ND | 不可抵消真实伤害 |
| FD | 晕眩（1~3回合） |
| WL | 遇障停止 |

## 🚀 技术栈

```
Python + FastAPI + WebSocket + 原生 HTML/CSS/JS
AI 向导：DeepSeek API
部署：Render.com
```

## 📁 项目结构

```
├── main.py              # FastAPI 服务器 + WebSocket
├── game_engine.py        # 游戏逻辑引擎
├── characters.py         # 角色技能数据
├── ai_opponent.py        # 单人模式 AI
├── templates/
│   ├── index.html        # 大厅（创建/加入房间/向导聊天）
│   └── game.html         # 对战界面
├── static/
│   ├── style.css         # 暗黑日系主题
│   ├── game.js           # 前端游戏逻辑
│   ├── player.js         # 玩家系统（ID+名称缓存）
│   ├── guide.js          # AI 向导（蝴蝶忍）
│   └── battle-records.js # 战斗记录存储
└── requirements.txt
```
